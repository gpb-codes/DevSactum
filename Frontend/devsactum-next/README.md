## Descripción

Proyecto base construido con Next.js utilizando create-next-app.  
Diseñado para desarrollo rápido, escalabilidad y despliegue optimizado.

---

## Características

- Next.js App Router
- Server Components
- Optimización automática
- Font optimization con next/font
- Deploy listo para Vercel
- Estructura escalable
